# Chrome Web Store 掲載情報（ドラフト）

デベロッパーダッシュボードの各入力欄にそのままコピーして使う想定。

## 拡張機能名

音声タブコントローラー

## 短い説明（132文字以内、一覧・検索結果に表示される）

音が出ているタブを一覧表示し、ワンクリックでミュート/解除。サイト単位の常時ミュートにも対応。

（78文字。日本語は1文字1バイト超のため、念のためダッシュボード上で文字数警告が出ないか確認すること）

## 詳細な説明

```
音が出ているタブが多すぎて、どこから音が鳴っているか分からなくなったことはありませんか？

音声タブコントローラーは、現在のウィンドウで音声を再生中・ミュート中のタブを一覧表示し、
ワンクリックでミュート・解除できるシンプルな拡張機能です。

■ 主な機能
・音声再生中のタブを一覧表示（再生中／再生中だがミュート済み／ミュート済みで無音、の3状態を区別）
・タブごとの個別ミュート／解除
・一括ミュートと、この拡張機能がミュートしたタブだけを対象にした安全な一括解除
　（ユーザーが手動でミュートしたタブや、他の拡張機能がミュートしたタブは解除しません）
・サイト単位の常時ミュート登録（登録したサイトは新しいタブでも自動的にミュートされます）

■ プライバシーについて
・閲覧履歴やページ内容を外部に送信することは一切ありません
・すべての処理はブラウザ内で完結します
・タブのタイトル・URLのホスト名は、一覧表示のためだけに一時的に使用し、保存はしません
　（常時ミュート設定として登録したホスト名のみ、お使いのブラウザに保存されます）

■ こんな方におすすめ
・調べ物や作業で多くのタブを開く方
・動画・音楽サイトを開いたまま忘れてしまうことが多い方
・特定のサイトは常に無音にしておきたい方
```

## 単一目的の説明（Single purpose、審査時に求められる場合がある）

このアプリケーションの唯一の目的は、ユーザーが開いているタブの音声再生状態を確認し、
ミュート/解除を管理することです。それ以外の機能（データ収集、広告表示、外部送信等）は
一切含みません。

## 権限の正当化理由（Permission justification）

### `tabs`

タブの音声再生状態（`audible`）とミュート状態（`mutedInfo`）を取得し、ポップアップに
一覧表示するために必要です。また、ユーザーの操作に応じて対象タブをミュート/解除する
（`chrome.tabs.update`）ためにも使用します。取得したタイトル・URLのホスト名は
一覧表示のためだけに一時的に使用し、外部への送信や永続的な保存は行いません
（常時ミュート設定として登録したホスト名のみ、`chrome.storage.sync`に保存されます）。

### `storage`

サイト単位の常時ミュート設定を`chrome.storage.sync`に保存するために必要です。
ブラウザ内にのみ保存され、外部サーバーへ送信されることはありません。

## スクリーンショット（要手動撮影）

Chrome Web Storeは1280x800または640x400のスクリーンショットを推奨（1〜5枚）。
以下を撮影することを推奨：

1. 複数タブが音声再生中の状態でpopupを開いた一覧画面
2. 一括ミュート実行後の画面（成功件数の表示）
3. 常時ミュートのサイト登録画面
4. 実際のタブ（スピーカーアイコンにミュート表示が出ている状態）

撮影方法：`chrome://extensions`で拡張機能を読み込んだ状態で、実際に
`e2e/fixtures/audio-page.html`等を開いて操作しながらスクリーンショットを撮る。
